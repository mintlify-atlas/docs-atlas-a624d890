title: hyperstack-interpreter
description: AST transformation runtime and VM for HyperStack streaming pipelines
---

The `hyperstack-interpreter` crate provides the core execution engine for processing Solana blockchain events into typed state projections.

<Info>
  View on [docs.rs](https://docs.rs/hyperstack-interpreter)
</Info>

## Installation

```toml
[dependencies]
hyperstack-interpreter = "0.2"
```

## Public Exports

### Types

<ResponseField name="Mutation" type="struct">
  Represents a state change with export name, key, patch data, and optional append operations
  
  ```rust
  pub struct Mutation {
      pub export: String,
      pub key: Value,
      pub patch: Value,
      pub append: Vec<String>,
  }
  ```
</ResponseField>

<ResponseField name="EventWrapper<T>" type="struct">
  Generic wrapper for event data with context metadata
  
  ```rust
  pub struct EventWrapper<T = Value> {
      pub timestamp: i64,
      pub data: T,
      pub slot: Option<u64>,
      pub signature: Option<String>,
  }
  ```
  
  Events captured with `#[event]` are automatically wrapped in this structure.
</ResponseField>

<ResponseField name="CaptureWrapper<T>" type="struct">
  Generic wrapper for account capture data with metadata
  
  ```rust
  pub struct CaptureWrapper<T = Value> {
      pub timestamp: i64,
      pub account_address: String,
      pub data: T,
      pub slot: Option<u64>,
      pub signature: Option<String>,
  }
  ```
  
  Accounts captured with `#[capture]` are automatically wrapped in this structure.
</ResponseField>
