---
title: hyperstack-typescript
description: Core TypeScript client for HyperStack streaming servers
---

The `hyperstack-typescript` package provides the core client implementation for connecting to HyperStack servers and consuming real-time entity updates in TypeScript/JavaScript applications.

<Info>
  View on [npm](https://www.npmjs.com/package/hyperstack-typescript)
</Info>

## Installation

```bash
npm install hyperstack-typescript
```

## Quick Start

```typescript
import { HyperStack } from 'hyperstack-typescript';
import { MyStack } from './generated/stack';

const client = await HyperStack.connect(MyStack, {
  autoReconnect: true,
});

for await (const entity of client.views.users.list.use()) {
  console.log('User:', entity);
}
```
